Furniro – E-Commerce Frontend Design

Overview


Furniro is a responsive e-commerce frontend design project built as part of my web development coursework.

This project demonstrates HTML, CSS, and JavaScript integration to create a functional product browsing and cart system. Users can view product listings, see detailed product pages, and add items to the cart with dynamic updates.

Features Implemented

1. Homepage / Product Listing

   Responsive layout displaying multiple products

   Product cards with image, name, price, and add-to-cart button

   Navigation header with home, shop, about, contact, and cart count

   Smooth integration with detail pages for each product

2. Product Detail Page

   Displays product name, image, price, description, and key features

   Dynamically loads product data based on selection from homepage

   Add-to-cart functionality directly from detail page

3. Shopping Cart

   Stores items in localStorage for persistence

   Users can add products, adjust quantities, and remove items

   Displays cart count in the header and subtotal/total in cart page

   Provides toast notifications for added products

Technologies Used

   HTML5 – Semantic markup for all pages

   CSS3 – Responsive styling with flexbox and custom styles

   JavaScript (ES6) – Dynamic rendering of products, cart management, and localStorage integration
 
   Font Awesome – Icons for navigation and cart

   Git/GitHub – Version control and weekly commits

How to Use

Clone the repository:

git clone https://github.com/m4857586/ecommerce-frontend-design.git

Open index.html in a web browser.

Browse products, click on a product to see details, and add items to the cart.

Cart data persists between sessions using localStorage.
